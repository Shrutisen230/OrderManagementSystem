# Order Management System 

### Description

This a Menu-driven console-based project for interactively perform CRUD , search and sort operations on the Order Management System using Java and HashSet .
It uses:

- HashSet for item storage (ensuring unique item IDs).

- Custom Exceptions to handle errors (e.g., duplicate items, item not found).

- Comparators to sort items by name and price.

- Menu-driven interface for interactive console usage.

It follows object-oriented principles with multiple classes, an interface, and exception handling.

---

### Features

- Add Items : Add new items with ID , name and price .
- View Items : Display all items in the order .
- Update Items : Update item details using Item ID .
- Delete Item : Delete items by Item ID .
- Search Item : Search item by name .
- Sort Item By Price : Sorting by Price .
- Sort Item By Name : Sorting by Name .

---

### Tools & Technologies 

- Language : Java 25
- IDE : IntelliJ IDEA
- Collections : HashSet
- Sorting : Custom Comparators (SortByName , SortByPrice)
- I/O : Console-based interaction
